PROJECT DATE: circa 2021

INPUTS:
LEFT -> Create a wall node
RIGHT -> Set node to empty

S + LEFT -> Create starting node
E + LEFT -> Create ending node

R -> Run the algorithm
C -> Clears the board and resets for next path
